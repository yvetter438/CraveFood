## Image Specifications

- **Format:** PNG or JPG (PNG preferred for transparency)
- **Aspect Ratio:** 9:16 (portrait/vertical - phone screen ratio)
- **Recommended Size:** 1080x1920px or 750x1334px
- **Max File Size:** Keep under 2MB for fast loading
- **Background:** Can be transparent for phone mockups

## Tips

1. Use phone mockup templates for a professional look
2. Show actual app interface when possible
3. For video mockups, export a high-quality frame
4. Ensure text/UI elements in mockups are readable
5. Consider using tools like Figma, Sketch, or online mockup generators

## Preview

The mockups will appear in phone-shaped containers with:
- Rounded corners (3xl border radius)
- White border (border-4)
- Shadow effect
- Semi-transparent backdrop

If no image is uploaded, a placeholder with the section icon will show.
